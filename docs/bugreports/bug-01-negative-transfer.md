# BUG-01. Приложение принимает отрицательную сумму перевода

## Окружение

- Приложение: F-Bank из `dist.zip`
- Браузер: Google Chrome
- URL: `/?balance=5000&reserved=0`

## Предусловия

- Открыта страница сервиса.
- Выбран счет `Рубли`.
- Введен корректный номер карты из 16 цифр.

## Шаги воспроизведения

1. Ввести в поле суммы значение `-1000`.
2. Проверить доступность кнопки `Перевести`.

## Фактический результат

Кнопка `Перевести` остается доступной, сервис позволяет инициировать перевод с отрицательной суммой.

## Ожидаемый результат

Отрицательная сумма должна блокироваться валидацией. Кнопка перевода не должна отображаться или должна быть недоступна, пользователь должен видеть сообщение об ошибке.
