# BUG-02. Приложение принимает номер карты длиннее 16 цифр

## Окружение

- Приложение: F-Bank из `dist.zip`
- Браузер: Google Chrome
- URL: `/?balance=5000&reserved=0`

## Предусловия

- Открыта страница сервиса.
- Выбран счет `Рубли`.

## Шаги воспроизведения

1. Ввести в поле номера карты значение `41111111111111112`.
2. Проверить, появился ли блок ввода суммы перевода.

## Фактический результат

Приложение принимает 17 цифр, отображает блок суммы и позволяет продолжить сценарий перевода.

## Ожидаемый результат

Поле номера карты должно принимать только 16 цифр. При вводе 17-й цифры значение должно быть отклонено или показана ошибка валидации.
