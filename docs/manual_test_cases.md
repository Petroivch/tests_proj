# Ручные тест-кейсы

## TC-01. Успешное отображение формы перевода для рублевого счета

- Предусловия: открыть приложение по адресу `/?balance=5000&reserved=500`.
- Шаги:
  1. Выбрать счет `Рубли`.
  2. Ввести номер карты `4111111111111111`.
- Ожидаемый результат: отображается поле ввода суммы, комиссия и кнопка перевода.
- Фактический результат: форма перевода отображается корректно.
- Статус: Passed.

## TC-02. Корректный расчет недостаточности средств

- Предусловия: открыть приложение по адресу `/?balance=1000&reserved=950`.
- Шаги:
  1. Выбрать счет `Рубли`.
  2. Ввести номер карты `4111111111111111`.
  3. Оставить сумму `1000`.
- Ожидаемый результат: сервис сообщает о недостаточности средств, кнопка перевода недоступна.
- Фактический результат: отображается сообщение `Недостаточно средств на счете`.
- Статус: Passed.

## TC-03. Успешная подготовка перевода при достаточном балансе

- Предусловия: открыть приложение по адресу `/?balance=5000&reserved=0`.
- Шаги:
  1. Выбрать счет `Рубли`.
  2. Ввести номер карты `4111111111111111`.
  3. Оставить сумму `1000`.

- Фактический результат: комиссия рассчитывается, кнопка доступна.

## TC-04. Запрет перевода с отрицательной суммой

- Предусловия: открыть приложение по адресу `/?balance=5000&reserved=0`.
- Шаги:
  1. Выбрать счет `Рубли`.
  2. Ввести номер карты `4111111111111111`.
  3. Ввести сумму `-1000`.
- Ожидаемый результат: отрицательная сумма отклоняется, кнопка `Перевести` не отображается, показывается сообщение об ошибке валидации.
- Фактический результат: сервис позволяет выполнить перевод с отрицательной суммой.
- Статус: Failed.
- Баг-репорт: `docs/bugreports/bug-01-negative-transfer.md`.

## TC-05. Запрет номера карты длиннее 16 цифр

- Предусловия: открыть приложение по адресу `/?balance=5000&reserved=0`.
- Шаги:
  1. Выбрать счет `Рубли`.
  2. Ввести номер карты `41111111111111112`.
- Ожидаемый результат: номер карты отклоняется как невалидный, блок суммы перевода не появляется.
- Фактический результат: приложение принимает 17 цифр и позволяет перейти к вводу суммы.
- Статус: Failed.
- Баг-репорт: `docs/bugreports/bug-02-card-length.md`.
